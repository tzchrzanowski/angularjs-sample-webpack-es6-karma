# AngularJS 1.5 webpack, babel, and karma example

This repository is an example of how to use AngularJS 1.5 with Webpack, Babel (ES6+), and Karma (Jasmine).

# Run it

- Requires Node 8.4.0+

Or with npm:

    npm install
    npm start
    npm test

# build it with minification:

    webpack --optimize-minimize
# eslint
    init setup:
        npx eslint --init

    run:
        npx eslint ./
